LARSON'S NYSUS CODE EXAMPLE

This code was created for Larson Long's application to Nysus Solutions. It serves no functional purpose.

HOW TO RUN:
1. run nysus_init.js
    -will require a mySQL host, username and password
2. run nysus_createDatabase.js
    -will require a mySQL host, username and password
    -this program was supposed to also initialize the database, but that duty was seperated into nysus_init.js
3. host nysus_css.css and nysus_ajax.php
    -nysus_ajax.php will require a mySQL host, username and password
4. host nysus_query.php
    -must have access to nysus_ajax.php and nysus_css.css
